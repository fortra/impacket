# Impacket Examples

Impacket Examples is the optional examples distribution for Impacket.

It installs the example support modules from `impacket.examples` together with
the example scripts from `examples/`. This package depends on
`impacket-core` and is intended for users who want the examples without using
the backwards-compatible full `impacket` package name.

## Install

Install the examples package and its `impacket-core` dependency:

```bash
python3 -m pip install impacket-examples
```

Install the full backwards-compatible distribution with examples:

```bash
python3 -m pip install impacket
```

Install only the library:

```bash
python3 -m pip install impacket-core
```
