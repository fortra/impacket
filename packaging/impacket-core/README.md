# Impacket Core

Impacket Core is the library-only distribution for Impacket.

It includes the protocol classes, structures, and helpers under `impacket/`
without installing the optional example scripts that live in the full
`impacket` package.

## Install

Install the library-only distribution:

```bash
python3 -m pip install impacket-core
```

Install the full backwards-compatible distribution with examples:

```bash
python3 -m pip install impacket
```

To add the optional example scripts on top of the core package:

```bash
python3 -m pip install impacket
```
